# Auto Clipboard

> 浏览器插件，选中文本自动复制到剪切板，同时自动破解网页禁止复制的限制。

## 功能

- **破解复制限制** — 自动解除网页禁止复制、禁止选中文本的限制
- **自动复制** — 选中文本即自动复制（可配置为手动复制）
- **按站点配置** — 可对单个网站单独配置是否启用自动复制
- **剪切板历史** — 查看历史复制记录，支持置顶常用内容
- **提示语自定义** — 复制成功提示语可拖动定位，支持自定义配色
- **右键菜单** — 支持通过右键菜单快速启用/禁用自动复制
